# 🌌 Pedro Miranda — Immersive Portfolio

Welcome to the source code of my personal portfolio! This project was built to showcase not only my professional experience in Cybersecurity, Software Development, and AI, but also my passion for creating immersive, high-performance web experiences.

## 🎯 Project Overview

This portfolio goes beyond a traditional static website. It leverages modern web technologies to create an engaging narrative through **scrollytelling**, **3D WebGL particle effects**, and **kinetic typography**. The design follows a modern approach with neobrutalist elements and a sleek dark mode aesthetics.

## 🛠️ Technologies Used

- **[Vite](https://vitejs.dev/)**: Next-generation frontend tooling for lightning-fast development and optimized production builds.
- **[TypeScript](https://www.typescriptlang.org/)** & **JavaScript (ES6+)**: For robust and dynamic logic.
- **[Three.js](https://threejs.org/)**: Used to render the interactive 3D WebGL particle scene that reacts to user interactions.
- **[GSAP (GreenSock)](https://gsap.com/)**: For complex timeline animations, scroll-triggered events, and kinetic typography.
- **[Lenis](https://lenis.studiofreight.com/)**: Implementing butter-smooth scrolling to enhance the scrollytelling experience.
- **HTML5 & Vanilla CSS**: Utilizing modern CSS properties, custom properties (variables), and grid/flexbox for responsive layouts without heavy framework overhead.

## ✨ Key Features

- **Interactive 3D Background**: A custom WebGL particle system built with Three.js that dynamically shifts based on scroll position and mouse movements.
- **Smooth Scrollytelling**: Powered by GSAP ScrollTrigger and Lenis, the content reveals itself organically as the user navigates down the page.
- **Custom Cursor**: A custom-designed cursor that interacts contextually with clickable elements.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewports, ensuring the immersive experience is never compromised.
- **Performance First**: Minimal dependencies and highly optimized assets to guarantee fast load times and a consistent 60fps experience.

## 🚀 Running Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/PedroMiranda243/pedro-portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd pedro-portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser** and visit `http://localhost:5173/` (or the port specified in your terminal).

## 📦 Building for Production

To create an optimized production build:

```bash
npm run build
```
This will generate a `dist` folder containing the minified assets ready to be deployed.
